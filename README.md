# ShadowRealms

ShadowRealms is a terminal-based dungeon crawler built in C++. Choose a hero, battle monsters, use campfires strategically, and try to survive through a procedurally generated dungeon to face the final boss.

## Features
- Four unique character classes with different stats and playstyles
- Turn-based combat system (attack, special, defend, flee)
- Randomized dungeon rooms and monster encounters
- XP, gold, leveling system, and upgrades
- Checkpoint/campfire system for healing and progression choices
